You may want to run the following command to target certain devices:

ionic emulate ios --target="iPhone-6"

# Valid values for "--target" (case sensitive):
#   iPhone-4s
#   iPhone-5
#   iPhone-5s
#   iPhone-6-Plus
#   iPhone-6
#   iPad-2
#   iPad-Retina
#   iPad-Air
#   Resizable-iPhone
#   Resizable-iPad
